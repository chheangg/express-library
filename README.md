# express-library
## Overview
A local library back-end app made with express & love! It can store books, book instances, your favorite authors, and even genres. You can do many thing with all of the forementioned objects too!

## Tech used
- JS
- NodeJS
- Express
- Pug/Jade
- Mongoose
- MongoDB

## Middleware used
- Express-validator
- Helmet
- Compressor
- Async
- moongoose
